# GCP実務ナビ

## テーマ
臨床試験におけるGood Clinical Practice（GCP）の実務知識を体系的に発信するブログ。ICH-GCPガイドラインの解釈から、査察対応、同意取得の実務まで、現場で役立つ情報を提供する。

## ターゲット読者
- 臨床開発モニター（CRA）、臨床開発担当者
- 治験施設のスタディコーディネーター（CRC）
- 治験責任医師・分担医師

## 想定記事カテゴリ
- ICH-GCPガイドライン逐条解説
- 治験モニタリングの実務Tips（SDV、リスクベースドモニタリング）
- 同意説明文書（ICF）作成と同意取得プロセス
- PMDA/FDA査察対応と準備ポイント
- GCP違反事例と再発防止策
